Interaction Design--Life Flower

This project is the final project of an interaction design course. My aim is to create a mechanical flower that opens and closes in response to the users heart rate, with the mechanical flower adorned on a clock face. The ephemeral butterfly element is also incorporated into this project. I have created a series of butterflies using Processing, which change color in response to the users heart rate in an interactive animation. The combination of various elements is intended to express an intuitive sense of the passage of time.
